# Environment Variables

This service uses the following environment variables:

- `MONGO_URI`: MongoDB connection string (default: mongodb://localhost:27017/game)
- `PORT`: TCP port for the microservice (default: 8877)
- `JWT_SECRET`: Secret key for JWT signing (default: REPLACE_WITH_A_STRONG_SECRET)
- `ADMIN_CREATION_SECRET`: Secret key for admin account creation (default: ADMIN12345)
- `REFRESH_TOKEN_TTL_DAYS`: Refresh token time-to-live in days (default: 7)
- `SESSION_INACTIVITY_MINUTES`: Admin session inactivity timeout in minutes (default: 60)
- `MAIL_HOST`: SMTP server host (default: smtp.gmail.com)
- `MAIL_PORT`: SMTP server port (default: 587)
- `MAIL_SECURE`: Use SSL for email (default: false)
- `MAIL_USER`: Email account for sending OTP (default: thepeacefulfarmstead@gmail.com)
- `MAIL_PASS`: App password for email account (default: slun vjvi opzm itac)
- `MAIL_FROM`: Sender name and email (default: FARMITY Admin <thepeacefulfarmstead@gmail.com>)
